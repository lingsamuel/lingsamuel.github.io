使用 Proton 在 Linux 下玩 Windows 游戏

Ling Samuel • 2020-07-07 • game  • proton  linux  game 

用 Proton1 在 Linux 底下跑了几个小游戏，感觉还不错。

例如 Crying Suns2，在 ProtonDB3 上被标记为 Platinum，兼容性还挺好的，没遇到什么问题。除了我的笔记本没有独显，帧率有点低以外，挺好。

实际上，在 ProtonDB 上大部分被标记为 Borked 的游戏都是大型 3D 游戏，例如《彩虹六号：围攻》，本来也不指望能在 Linux 底下跑。
奇怪的是 DOTA 2 和 CS: GO 的兼容等级还被标记为 Gold，但它们都有 Linux Native 版的。不是很懂。

从 ProtonDB 来看，高兼容性的游戏非常多，而这对开发者而言是无感知的。不得不说 Valve 还是下了颇大的功夫。这就是专注于推动游戏进步（？）的公司吗？太强了。
因为没有独显，跑个小独立游戏帧率都不咋地，也不能确定性能损失到底几何。不过，能跑小游戏就差不多了。

安装方法

无需配置，pacman 安装 Proton，然后在 Steam 客户端的 Settings - Steam Play 里启用即可。如果不是 Archlinux 的话，启用 Steam Play 后，Library 里会自动添加各版本的 Proton，也许可以从这里安装，没试过，不知道。
